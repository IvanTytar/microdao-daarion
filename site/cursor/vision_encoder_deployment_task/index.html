    Problem: Container fails to start¶

    Check:

    docker-compose logs vision-encoder

    Common issues: 1. CUDA not available ‚Üí Check nvidia-smi and Docker GPU runtime 2. Model download fails ‚Üí Check internet connection, retry 3. OOM (Out of Memory) ‚Üí Use smaller model (ViT-B-32) or check GPU memory

    Problem: Slow inference¶

    Check device:

    curl http://localhost:8001/health | jq '.device'

    If "device": "cpu" ‚Üí GPU not available, fix NVIDIA runtime

    Problem: Qdrant not accessible¶

    Check:

    docker-compose ps qdrant
docker exec -it dagi-vision-encoder ping qdrant

    Restart:

    docker-compose restart qdrant
